# 🎓✨ Sulekha Zenith Academy

> 🌟 **Personal Guidance • Strong Foundation • Bright Future**

A modern, responsive, and feature-rich educational platform designed for **Sulekha Zenith Academy**, a trusted home-based coaching institute dedicated to academic excellence and personalized learning.

## 🚀 Features

📚 **Course Management**
Explore subjects, batches, and learning programs with detailed information.

⭐ **Student Reviews & Ratings**
Share learning experiences and rate teaching quality through an interactive review system.

📧 **Direct Contact & Email Integration**
Visitors can easily connect with the educator through contact forms and direct email support.

💬 **WhatsApp Integration**
Instant communication with one-click WhatsApp messaging.

🗺️ **Google Maps Integration**
Locate the academy quickly with embedded maps and navigation support.

🖼️ **Gallery Management**
Showcase classroom activities, achievements, and educational events.

🎯 **Demo Class Booking**
Allow prospective students to request and schedule demo classes online.

🔒 **Hidden Secure Admin Dashboard**
Exclusive admin access with content management, enquiry tracking, review moderation, and website customization tools.

🌙 **Light & Dark Mode**
Modern user experience with seamless theme switching.

📱 **Fully Responsive Design**
Optimized for desktop, tablet, and mobile devices.
